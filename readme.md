# URL Shortener

A full-stack URL shortener application built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js). This project allows users to generate short URLs from long links, track usage, and manage link expirations.

---

## Features

- **Shorten URLs:** Convert long URLs into short, easy-to-share links.
- **Auto-expiry:** Short URLs can be set to expire automatically to reduce database clutter.
- **Frontend & Backend Separation:** Clean architecture with React.js frontend and Node.js + Express backend.
- **MongoDB Storage:** Efficient storage of URL mappings and metadata.

---

## Tech Stack

- **Frontend:** React.js, HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Tools:** Git, VS Code, Postman
